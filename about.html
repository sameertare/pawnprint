ℹ About OpenFile

A local-first chess toolkit for players, parents, and tournament organizers.

What this is

OpenFile is a set of seven small, focused chess tools — deep game analysis, a live engine board, Swiss tournament pairings, an opening tree, a report comparer, and rating estimators — each its own page, reachable from the home hub. There's no single sprawling app to learn; open the tool you need for the task in front of you.

Runs entirely in your browser

Every tool runs client-side. Game analysis uses Stockfish 18 (lite) compiled to WebAssembly and executed on your own machine — your PGNs and reports are never uploaded anywhere. Live games are streamed directly from the public lichess API straight to your browser; there's no OpenFile server in that path. Swiss tournaments persist to your browser's local storage. The only feature that touches a backend at all is the optional server-side report save/load, and even that's off by default on the static-hosted version of the site.

Install & use offline

OpenFile is an installable PWA. Most browsers offer an install prompt (address-bar icon, or browser menu → "Install OpenFile"); installed, it opens in its own window with no browser chrome. A service worker caches the app shell and the Stockfish engine files after your first visit, so position analysis and PGN review keep working with no connection at all once the engine has loaded — handy in a tournament hall with bad wifi. Live game streaming and account lookups still need a connection, since those come from lichess's/chess.com's public APIs.
